# Playwright
Решение: [playwright_part](playwright_part)  
Сопроводительный readme к этой части: [readme.md](readme.md)

# Нужно написать набор тестов для любого поисковика.
- поиск по тексту;
- поиск по изображению;
- позитивные сценарии;
- негативные сценарии.

# Отчетность
- Allure report с шагами и артефактами;
- возможность посмотреть выполнение теста по шагам (Trace Viewer аттачем к
каждому тесту).

# Требования к решению
- проект должен запускаться по инструкции;
- структура и код должны быть читаемыми;
- тесты должны быть стабильными;
- результат выполнения должен быть воспроизводим.
